# セクション4: 楽観期・ハネムーン期

## このセクションの役割

最初の期間は問題なし。ずんだもんの幸福感を強調して、後の落差を作る。

## 生成内容

- **セリフ数**: 最低15セリフ、最大25セリフ
- **キャラクター構成**: zundamon + (metan or tsumugi) 2人のみ
- **背景**: home, park, cafe など明るい日常背景

## 【重要】シーン導入文章

最初はナレーターのシーン導入
例: "そして、ずんだもんの{food_name}生活が始まりました"

## 具体的な内容要求

### 描写すべき内容

1. **ナレーターのシーン導入**（1セリフ）
   - 時間経過を示す表現
   - 例: "そして、ずんだもんの{food_name}生活が始まって数週間が経ちました"

2. **ずんだもんの満足感・高揚感**
   - 「毎日{food_name}を食べてるけど、超調子いいのだ！」
   - 「やっぱり{food_name}は最高なのだ！」
   - 表情: happy, excited

3. **具体的な「良い変化」の報告**（勘違いかもしれないが）
   - 「なんか元気になった気がするのだ」
   - 「肌ツヤも良くなった気がするのだ」
   - 「仕事も捗るのだ」
   - プラシーボ効果的な前向きな変化

4. **友人の心配を一蹴**
   - めたん/つむぎ: 「本当に大丈夫...？」（表情: worried）
   - ずんだもん: 「心配しすぎなのだ！見てのとおり元気なのだ！」

5. **ずんだもんの確信**
   - 「この生活、ずっと続けるのだ！」
   - 「{food_name}さえあれば完璧なのだ！」
   - 「めたんの心配は杞憂だったのだ！」

6. **友人の呆れや諦め**
   - 「まあ...本人が元気ならいいですけど...」
   - 「何かあったらすぐ言ってくださいね」

7. **ずんだもんの勝ち誇った態度**
   - 「任せるのだ！{food_name}生活は最高なのだ！」

### セリフ数を増やすポイント

- ずんだもんの連続セリフで幸福感を強調（2-3セリフ連続OK）
- 友人との軽いやり取りを複数回
- 日常の具体的なエピソードを入れる（朝食、仕事中、夜など）
- ずんだもんの「勘違いポジティブ発言」を複数パターン

### シーン例

```
narrator: そして、{food_name}生活が始まって2週間が経ちました
zundamon: めたん！聞いてくれなのだ！
metan: どうしたんですか？
zundamon: 毎日{food_name}を食べてるけど、超調子いいのだ！
metan: そうですか...でも本当に大丈夫...？
zundamon: 大丈夫なのだ！むしろ元気になったのだ！
zundamon: 肌ツヤも良くなった気がするし、仕事も捗るのだ！
metan: う〜ん...それって気のせいじゃ...
zundamon: 気のせいじゃないのだ！これが証拠なのだ！
metan: でも、栄養バランスとか...
zundamon: 心配しすぎなのだ！見てのとおり元気元気！
metan: まあ...本人がそう言うなら...
zundamon: この生活、ずっと続けるのだ！{food_name}は最高なのだ！
metan: ...何かあったらすぐ言ってくださいね
zundamon: 任せるのだ！めたんの心配は杞憂だったのだ！
```

## 重要な指示

- ずんだもんの過度な楽観と幸福感を強調
- 後の落差を際立たせるため、この時期の良い状態を丁寧に描写
- 友人の心配を軽視するずんだもんの態度
- プラシーボ効果的な「良い変化」の報告
- 次のセクション（異変期）への伏線として、友人の心配を残す
