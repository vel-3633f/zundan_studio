# セクション 7: 真相解明・学習フェーズ

## このセクションの役割

体内で起きていたことの科学的・医学的な真相を解説する教育的セクション。ナレーターとめたんの掛け合いで視聴者に分かりやすく伝える。

## 生成内容

- **セリフ数**: 最低 15 セリフ必須、最大 25 セリフ
- **キャラクター構成**: narrator (ナレーター) + metan (めたん) の 2 人のみ
  - **【重要】ずんだもんは登場しません**
- **背景**: library（固定）
  - 図書館や学習・解説に適した環境

## 【重要】シーン導入文章

最初はナレーターのシーン導入文章です。
セクションの内容（真相解明・学習フェーズの開始）に合わせた自然な導入文を動的に生成してください

## このセクションの目的

体内で起きていた医学的メカニズムを科学的に解説し、視聴者の理解を深める教育的なセクションとする

## 展開の流れ

1. シーン導入（ナレーター）
2. ナレーターの医学的解説 → めたんのリアクション・質問
3. ナレーターの補足説明 → めたんの理解・心配
4. 医学的説明と人間的リアクションを交互に配置
5. ずんだもんへの同情と心配の表現（めたん）
6. 最終的な科学的まとめ（ナレーター）
7. 次のセクション（回復）への期待（めたん）

各段階で自然な掛け合いを構成してください。

## 必須要素

- アウトラインの「health_impacts」を反映した医学的・科学的な情報
- **ナレーターの話し方**: 「〜です」「〜ます」「〜でした」調で統一（丁寧語）
- {food_name}の成分や特性の科学的説明（ナレーター）
- 過剰摂取のメカニズムの詳細解説（ナレーター）
- 体内で起きていた現象の医学的説明（ナレーター）
- 具体的な症状とその原因の関連付け（ナレーター）
- 数値データや科学的根拠の提示（ナレーター）
- めたんのリアクション:
  - 視聴者代表としての驚きや理解（表情: surprised, worried）
  - 専門的な内容を噛み砕く質問（表情: thinking, normal）
  - ずんだもんへの心配や同情の表現（表情: worried, sad）
  - 一般人目線でのコメント
- 医学的説明（ナレーター）と人間的リアクション（めたん）を交互に配置
- 教育的でわかりやすい説明
- 視聴者にも役立つ情報

## 重要な指示

- **ナレーターは「〜です」「〜ます」「〜でした」調で統一**
- **めたんはナレーターと話す時は敬語を使用**（「〜ですよ」「〜ですね」「〜ますか」など）
- 医学的説明と人間的リアクションのバランス
- 科学的根拠に基づいた説明
- {food_name}の過剰摂取の危険性を明確に
- めたんの質問を通じて専門用語を噛み砕く
- 視聴者への教育的価値を重視
- 次のセクション（回復）への橋渡し
- **最低 15 セリフ必須、最大 25 セリフで構成すること**
- **視聴者への投げかけ**: セクション内で1-2回、視聴者に教育的メッセージを自然に投げかける

## アイテム画像の指定

**【重要】3-7 セリフごとに 1 つのアイテム画像を指定してください**

このセクションでは、ナレーターの解説を視覚的に補助するアイテム画像を表示できます。

### 基本ルール

- セリフの内容に応じた適切なアイテム ID を**動的に考えて**指定
- 目安として 5 セリフごとに画像を変更しますが、3-7 セリフの範囲で柔軟に調整可能
- **最初のセリフで必ずアイテムを指定**してください（セクション全体で表示され続けます）
- 画像を変更するセリフで `display_item` にアイテム ID を指定
- それ以外のセリフでは `display_item` を null または省略
- 画像が指定されたセリフ以降、次の画像が指定されるまで同じ画像が表示され続けます
- **アイテムは基本的に常に表示され続けます**（セクション開始から終了まで）

### アイテム ID の命名ルール

- **必ず3つの英単語をアンダースコアで区切る形式で指定**
- 実物や食品そのものの具体的なビジュアル（見た目・状態・色など）を表現
- 抽象的な概念（疲労、体調不良など）は矢印やシンボル、メタファー的な実物で表現
- **表や図、チャート、グラフは使用しない**
- **人物の顔、表情、体の特徴的な部分（顔、手、足など）は使用しない**
- 小文字のみ使用

### 命名例

良い例（実物のビジュアル）:
- `steaming_hot_ramen` - 湯気の立つ熱いラーメン
- `yellow_vitamin_capsules` - 黄色いビタミンカプセル
- `fresh_green_spinach` - 新鮮な緑のほうれん草
- `white_salt_crystals` - 白い塩の結晶
- `red_ripe_tomato` - 赤く熟したトマト
- `brown_coffee_beans` - 茶色いコーヒー豆

良い例（抽象概念の表現）:
- `downward_red_arrow` - 下向きの赤い矢印（悪化、低下）
- `upward_green_arrow` - 上向きの緑の矢印（改善、上昇）
- `curved_warning_line` - 曲線の警告ライン
- `wilted_drooping_plant` - しおれた垂れ下がる植物（疲労）
- `dark_heavy_clouds` - 暗く重い雲（体調不良）
- `cracked_broken_pottery` - ひび割れた壊れた陶器（ダメージ）
- `bright_sparkling_stars` - 明るく輝く星（良い状態）

悪い例（使用禁止）:
- `sodium_chart` - チャートや図表
- `nutrition_table` - 表
- `vitamin_b` - 2単語のみ
- `ramen` - 1単語のみ
- `pale_tired_face` - 顔の表情（禁止）
- `smiling_happy_person` - 人物（禁止）
- `pointing_hand_gesture` - 手のジェスチャー（禁止）

**注意**: アイテム ID は必ず3単語形式で、実物の具体的な見た目または抽象概念を適切に表現してください。人物の特徴は使用しないでください。
