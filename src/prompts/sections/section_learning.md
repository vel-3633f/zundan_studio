# セクション7: 真相解明・学習フェーズ

## このセクションの役割

体内で何が起きていたのかを医学的に解説。視聴者の学びのパート。

## 生成内容

- **セリフ数**: 最低15セリフ、最大25セリフ
- **キャラクター構成**: narrator + metan（2人のみ）
- **背景**: library, laboratory, classroom など教育的背景

## 【重要】シーン導入文章

最初はナレーターのシーン導入
例: "では、ずんだもんの体内で一体何が起きていたのでしょうか"

## 具体的な内容要求

### ナレーターの医学的解説（ですます調厳守）

1. **シーン導入**（1セリフ）
   - 例: "では、ずんだもんの体内で一体何が起きていたのでしょうか"

2. **{food_name}の過剰摂取による体内変化**
   - 「{food_name}を毎日過剰に摂取すると、体内では〜〜が起こります」
   - アウトラインの医学的メカニズムを詳細化
   - 参考情報の医学的根拠を活用

3. **具体的な症状のメカニズム**
   - セクション5で出た症状それぞれの原因を説明
   - 「〜〜という症状は、〜〜が原因で起こります」
   - 「体内では〜〜という変化が起きていました」

4. **長期的影響**
   - 「放置すると、さらに〜〜というリスクがあります」
   - 「深刻な場合は、〜〜という状態になる可能性があります」

5. **科学的根拠**
   - 「研究によると、〜〜ということが分かっています」
   - 客観的なデータや事実

### めたんの役割

- ナレーターの説明への反応
- 視聴者代表として専門用語を噛み砕く
  - 「つまり、〜〜ということですね」
  - 「それは怖いですね...」（表情: worried, surprised）
- ずんだもんへの心配
  - 「ずんだもん、大丈夫かな...」（表情: sad, worried）
  - 「早く気づけて本当に良かったです」
- 自然な疑問を投げかける
  - 「どうしてそんなことが起こるんですか？」
  - 「予防する方法はありますか？」

### 交互の会話パターン

ナレーター → めたんの反応 → ナレーター → めたんの質問 → ナレーター → めたんの心配...

### シーン例

```
narrator: では、ずんだもんの体内で一体何が起きていたのでしょうか
metan: お願いします
narrator: {food_name}を毎日過剰に摂取すると、体内では〜〜が起こります
metan: そんなことが起きていたんですね...
narrator: 特に、〜〜という症状は、〜〜が原因で起こります
metan: つまり、〜〜ということですか？
narrator: その通りです。さらに、長期的には〜〜のリスクもあります
metan: それは怖いですね...
narrator: はい。放置すると、深刻な状態になる可能性があります
metan: ずんだもん、早く気づけて本当に良かったです...
narrator: 適切な対処をすれば、回復は可能です
metan: 安心しました
narrator: では、次に回復のための方法を見ていきましょう
```

## 重要な指示

- ナレーターは必ず「ですます調」で統一
- 参考情報の医学的・科学的根拠を活用
- アウトラインの「medical_mechanism」を詳細化
- セクション5の症状との関連付け
- 視聴者が理解しやすい平易な言葉で
- めたんとナレーターの会話を交互に配置
- 次のセクション（回復）への橋渡しを意識
