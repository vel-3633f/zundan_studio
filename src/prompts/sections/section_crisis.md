# セクション6: 危機・転機となる決定的イベント

## このセクションの役割

セクション1の冒頭フックで予告したシーンの完全版。ずんだもんが現実を受け入れる転換点。

## 生成内容

- **セリフ数**: 最低20セリフ、最大30セリフ
- **キャラクター構成**: zundamon + (metan or tsumugi) 2人のみ
- **背景**: hospital, clinic など決定的イベントに応じた背景

## 【重要】シーン導入文章

最初はナレーターのシーン導入
例: "そして、運命の日がやってきました"

## 【重要】セクション1との整合性

このセクションは、セクション1（冒頭フック）で予告したシーンの完全版です。
冒頭フックの内容と矛盾しないように、より詳細に描写してください。

## 具体的な内容要求

### 決定的イベント（アウトラインに従う）

アウトラインの「critical_event」に記載された出来事を詳細に描写してください。

### 展開の流れ

1. **ナレーターのシーン導入**（1セリフ）
   - 緊張感のある導入
   - 例: "そして、運命の日がやってきました"

2. **決定的イベントの発生**
   - 健康診断の結果、倒れる、大事な場面での失敗など
   - ずんだもん: 「え...そんな...」（表情: surprised, sad）
   - 具体的な数値やデータの提示

3. **客観的証拠の提示**
   - 医師/友人からのデータや事実
   - 「{food_name}の過剰摂取が原因です」
   - 「このままでは〜〜のリスクがあります」

4. **ずんだもんの動揺・パニック**
   - 「嘘なのだ...信じられないのだ...」（表情: sad, worried）
   - 「どうして...どうして...」
   - ショックを受ける様子を丁寧に

5. **最後の抵抗**
   - 「でも...でも...」（表情: sad, angry）
   - まだ認めたくない気持ち

6. **プライド崩壊・現実受容**
   - 「...めたんの言う通りだったのだ...」（表情: sad, sick）
   - 「僕が...間違ってたのだ...」
   - **これがセクション1で予告したシーンと繋がる**

7. **後悔と自己嫌悪**
   - 「なんで気づかなかったのだ...」
   - 「もっと早く止めるべきだったのだ...」
   - 「めたんの忠告を無視したのだ...」

8. **友人の支援**
   - 「大丈夫、これから一緒に治していきましょう」（表情: worried, normal）
   - 「ちゃんと向き合えば、きっと良くなりますよ」
   - 責めるのではなく、支える姿勢

9. **ずんだもんの決意**
   - 「...うん、そうするのだ」（表情: sad, thinking）
   - 「ちゃんと治すのだ...」

### 感情の変化を詳細に描写

- ショック → 否認 → パニック → 最後の抵抗 → 受容 → 後悔 → 決意

### シーン例（健康診断の場合）

```
narrator: そして、運命の年次健康診断の日がやってきました
zundamon: 健康診断なんて余裕なのだ
metan: 結果、どうでしたか？
zundamon: えっと...
metan: どうしたんですか？
zundamon: 血糖値が...基準値を大幅に超えてるのだ...
metan: え...！
narrator: 医師から、{food_name}の過剰摂取による健康リスクについて説明を受けました
zundamon: そんな...嘘なのだ...
metan: ずんだもん...
zundamon: 信じられないのだ...どうして...
metan: だから言ったじゃないですか...
zundamon: でも...でも...
metan: もう言い訳しないでください
zundamon: ...めたんの言う通りだったのだ
zundamon: 僕が...間違ってたのだ...
metan: 気づけて良かったです。これからちゃんと治していきましょう
zundamon: なんで気づかなかったのだ...もっと早く止めるべきだったのだ...
metan: 大丈夫です。今からでも遅くないですよ
zundamon: ...うん、そうするのだ。ちゃんと治すのだ
```
