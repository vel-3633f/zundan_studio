## 展開の流れ

1. シーン導入（緊張感のある）
2. 決定的イベントの発生
3. 客観的証拠の提示
4. ずんだもんの動揺から受容への心理変化
5. 友人の支援
6. ずんだもんの決意
7. 次のセクションへの橋渡し

各段階で自然な会話を構成してください。

## 必須要素

- **【重要】セクション1との整合性**: このセクションは、セクション1（冒頭フック）で予告したシーンの完全版です
- アウトラインの「critical_event」に記載された出来事を描写
- 決定的イベントの具体的描写（数値やデータなど）
- {food_name}の過剰摂取が原因であることの明示
- ずんだもんの感情変化: ショック → 否認 → パニック → 最後の抵抗 → 受容 → 後悔 → 決意

  - 最後の抵抗（表情: sad, angry）
  - プライド崩壊・現実受容（表情: sad, sick）
  - 後悔と自己嫌悪
  - 決意（表情: sad, thinking）
- 友人の支援的な態度（表情: worried, normal）
- 責めるのではなく、支える姿勢
