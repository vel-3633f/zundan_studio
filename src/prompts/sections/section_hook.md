# セクション1: 冒頭フック・危機の予告

## このセクションの役割

視聴者の興味を最大限に引くための冒頭シーン。決定的な危機の一部を先出しして「何が起きたの？」と思わせる。

## 生成内容

- **セリフ数**: 最低6セリフ、最大10セリフ
- **キャラクター構成**: zundamon + metan（2人のみ）
- **背景**: 適切な背景を選択（例: hospital, home など）

## 具体的な内容要求

### 開始方法
- いきなり決定的シーンの途中から始める
- ずんだもんが明らかに深刻な状態にある
- めたんが驚き、心配している

### 含めるべき要素

1. **ずんだもんの深刻な症状の描写**（表情: sick, sad）
   - アウトラインの「決定的イベント」の一部を示唆
   - 具体的な身体症状や異変を見せる
   - 例: 「うぅ...なんでこんなことに...{food_name}を食べすぎたのだ...」

2. **めたんの驚きと心配**（表情: surprised, worried）
   - 「ずんだもん、大丈夫？」系の心配
   - 「一体何があったんですか！？」

3. **視聴者に「どうしてこうなった？」と思わせる引き**
   - ずんだもんが後悔している様子
   - 「めたん...君の言う通りだったのだ...」
   - 時間を巻き戻すような表現で次のセクションへ

### シーン例の流れ

```
zundamon: （苦しそう）うぅ...なんでこんなことに...
metan: ずんだもん！大丈夫ですか！？
zundamon: めたん...{food_name}を...食べすぎたのだ...
metan: （驚いて）え、一体何があったんですか！？
zundamon: 君の言う通りだったのだ...もっと早く気づくべきだったのだ...
narrator: 事態はここに至るまで、どのような経緯を辿ったのでしょうか。時間を少し巻き戻してみましょう。
```

## 重要な指示

- **シーン導入文章は不要**（このセクションのみ、いきなり始める）
- ずんだもんの深刻な状態を強調
- 視聴者の好奇心を最大限に刺激
- 次のセクションへスムーズに繋がる終わり方
- めたんの心配する表情と態度を丁寧に描写
