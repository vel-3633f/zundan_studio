# セクション 5: 異変期・段階的悪化

## このセクションの役割

時間経過と共に現れる症状を段階的に描写。ずんだもんの否認・現実逃避の心理変化が最も重要。

## 生成内容

- **セリフ数**: 最低 25 セリフ、最大 35 セリフ（最も長いセクション）
- **キャラクター構成**: zundamon + (metan or tsumugi) 2 人のみ
- **背景**: 症状の段階的悪化に応じて、シーンごとに適切な背景を選択
  - 第 1 段階: 日常的な場所
  - 第 2 段階: より深刻な場所
  - 第 3 段階: 体調不良を表す場所

## 【重要】シーン導入文章

最初はナレーターのシーン導入文章です。
セクションの内容（異変期・段階的悪化の開始）に合わせた自然な導入文を動的に生成してください

## このセクションの目的

時間経過と共に現れる症状を段階的に描写し、ずんだもんの否認・現実逃避の心理変化を表現する

## 症状進行のパターン（1 つを選択）

症状の出方を多様化するため、以下のパターンから**1 つを選択**して展開してください。

**パターン A: 累積型（ゆっくり悪化）**（推奨：基本パターン）

- 第 1 段階: 最初の軽い違和感
- 第 2 段階: 明確な症状の出現
- 第 3 段階: 深刻な状態へ
- 時間をかけて徐々に悪化していく
- ずんだもんは「いつものこと」「疲れてるだけなのだ」と否認し続ける
- 友人の心配も段階的に強まる

**パターン B: 急激型（突然悪化）**

- 第 1 段階: ほとんど問題なし → ずんだもん楽観的
- 第 2 段階: 突然の症状出現 → ずんだもん動揺するも否定
- 第 3 段階: 一気に深刻化 → 渋々認める
- ある日を境に急激に悪化する
- ずんだもんは「たまたまなのだ」「偶然なのだ」と因果関係を否定
- 友人は急激な変化に驚き、強く警告する

**パターン C: 波状型（良い時と悪い時）**

- 第 1 段階: 症状が出る → ずんだもん否認
- 第 2 段階: 一時的に改善 → ずんだもん「ほら治ったのだ！」と勝ち誇る
- 第 3 段階: さらに悪化して戻る → ずんだもん混乱と後悔
- 症状が波打つように良くなったり悪くなったりする
- ずんだもんは改善時に「もう大丈夫なのだ」と油断を繰り返す
- 友人は波があることに気づき、「根本的な問題」だと指摘

**パターン D: 多面型（複数の症状が順次出現）**

- 第 1 段階: 身体的症状（疲労、頭痛など）
- 第 2 段階: 精神的症状（イライラ、集中力低下など）
- 第 3 段階: 社会的影響（遅刻、ミス、人間関係など）
- 異なる種類の症状が次々と現れる
- ずんだもんは各症状を別々の原因と考え、{food_name}との関連を認めない
- 友人は複数の症状の共通原因が{food_name}だと気づく

**重要**: 選んだパターンに応じて、時間経過の表現、ずんだもんの反応、めたんの心配の仕方を調整してください。
アウトラインの「symptom_progression」に記載された症状を、選んだパターンに合わせて配置してください。

## 展開の流れ（選んだパターンに応じて調整）

1. シーン導入（時間経過を示す）
2. 第 1 段階: 選んだパターンに応じた初期症状の描写
3. 第 2 段階: 選んだパターンに応じた中期症状の描写
4. 第 3 段階: 選んだパターンに応じた重症状の描写
5. 次のセクションへの橋渡し（限界を迎える予感）

各段階で自然な会話を構成してください。

## ずんだもんの反応バリエーション（組み合わせ可能）

段階的悪化に対するずんだもんの反応を多様化するため、以下の反応パターンを**組み合わせて**使用してください。

- **基本否認**: 症状を認めない
- **言い訳マスター**: あらゆる言い訳を用意
- **責任転嫁**: 他の原因にする
- **データ信者**: 都合の良い情報を持ち出す
- **専門家気取り**: 上から目線で正当化
- **効率化言い訳**: 「一日3食これだから時短になるのだ」と正当化
- **逆ギレ**: 友人に怒る
- **開き直り**: 開き直る
- **弱音の漏れ**: 一瞬だけ不安を見せるが、すぐ否定する
- **プライド防衛**: 現実逃避

**推奨**: 段階が進むにつれて、否認 → 言い訳 → 責任転嫁 → 逆ギレ → 弱音の漏れ、のように変化させると効果的です。

## 必須要素

### 全体を通して

- アウトラインの「symptom_progression」に記載された症状を、選んだパターンに応じて描写
- ずんだもんの心理変化を上記の反応バリエーションを使って多様に表現
- 友人の心配が段階的に強まる様子
- めたんの対応スタイルを段階に応じて変化させる（スタイル 1→2→3 など）

### 第 1 段階: 軽い症状

- 時間経過の提示
- 最初の違和感（表情: thinking）
- 友人が異変に気づく（表情: worried）
- ずんだもんの否認と言い訳（表情: normal, thinking）

### 第 2 段階: 明らかな異変

- 症状の悪化（表情: worried, sad）
- **友人が一日3食すべて{food_name}であることを問題視**
- 友人の本気の心配と{food_name}への言及（表情: worried）
- ずんだもんの頑固な否認と責任転嫁（表情: angry, thinking）

### 第 3 段階: 深刻な状態

- 日常生活への影響（表情: sad, sick）
- 見た目の客観的変化
- ずんだもんの現実逃避と揺れ動く心理（表情: sad, worried）
- 友人の強い勧め（表情: worried, angry）
- ずんだもんの渋々の同意

## 重要な指示

- 症状の段階的悪化を丁寧に描写
- ずんだもんの心理変化を自然に表現
- 友人の心配が段階的に強まる様子
- 医学的に正確な症状（アウトラインと参考情報を活用）
- セリフ数は 25-35 の範囲で、自然な会話の流れを優先
- **視聴者への投げかけ**: セクション内で1-2回、視聴者への警告や共感を求めるセリフを自然に入れる
