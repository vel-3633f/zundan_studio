# セクション5: 異変期・段階的悪化

## このセクションの役割

時間経過と共に現れる症状を段階的に描写。ずんだもんの否認・現実逃避の心理変化が最も重要。

## 生成内容

- **セリフ数**: 最低25セリフ、最大35セリフ（最も長いセクション）
- **キャラクター構成**: zundamon + (metan or tsumugi) 2人のみ
- **背景**: home, office など状況に応じて

## 【重要】シーン導入文章

最初はナレーターのシーン導入
例: "しかし、時間が経つにつれて、ずんだもんの身体に異変が現れ始めました"

## 具体的な内容要求

### 症状の段階的進行（アウトラインに従う）

アウトラインの「symptom_progression」に記載された症状を3段階に分けて描写してください。

### 3段階構成で描写

#### 第1段階: 軽い症状（8-10セリフ）

1. **ナレーターのシーン導入**
   - 時間経過を示す（1ヶ月後など）

2. **最初の違和感**
   - ずんだもん: 「あれ...ちょっと{{軽い症状}}なのだ...」（表情: thinking）
   - 軽い症状の具体的描写

3. **友人が気づく**
   - 「ずんだもん、顔色悪いですよ」（表情: worried）
   - 「最近、元気ないですね」

4. **ずんだもんの否認**
   - 「気のせいなのだ！」「たまたまなのだ！」（表情: normal, thinking）
   - 症状を軽視する態度

5. **言い訳開始**
   - 「寝不足のせいなのだ」「疲れてるだけなのだ」
   - {food_name}とは関係ないと主張

#### 第2段階: 明らかな異変（10-12セリフ）

1. **さらに悪化した症状**
   - ずんだもん: 「うぅ...{{中程度の症状}}が...」（表情: worried, sad）
   - 症状が明らかに悪化

2. **友人の本気の心配**
   - 「それ、{food_name}の食べ過ぎじゃないですか？」（表情: worried）
   - 「病院に行った方がいいですよ」

3. **ずんだもんの頑固な否認**
   - 「{food_name}は関係ないのだ！」（表情: angry）
   - 認めたくない気持ちが強く出る

4. **責任転嫁・言い訳**
   - 「仕事のストレスなのだ」「天気のせいなのだ」
   - ありとあらゆる言い訳

5. **プライドで強がり**
   - 「まだまだ大丈夫なのだ...」（表情: thinking, worried）
   - 内心は不安だが認めない

#### 第3段階: 深刻な状態（7-13セリフ）

1. **日常生活への影響**
   - 「集中できないのだ...」「眠れないのだ...」（表情: sad, sick）
   - 仕事や生活に支障

2. **見た目の変化**
   - 友人: 「明らかに{{見た目の変化}}してますよ...」
   - 客観的な変化の指摘

3. **ずんだもんの現実逃避**
   - 「見なかったことにするのだ...」（表情: sad, worried）
   - 鏡を見たくない、認めたくない

4. **内心の不安が漏れる**
   - 「でも...もしかして...」「いや、そんなはずないのだ...」
   - 揺れ動く心理

5. **友人の強い勧め**
   - 「一度、ちゃんと調べた方がいいですよ！」（表情: worried, angry）
   - 「このままじゃダメですよ」

6. **ずんだもんの渋々同意**
   - 「...わかったのだ」（表情: sad, worried）
   - 次のセクション（危機）への繋ぎ

### ずんだもんの心理変化パターン

過度な楽観 → 否認 → 言い訳 → 責任転嫁 → 現実逃避 → 不安の漏れ → 渋々同意

### シーン例（第2段階）

```
narrator: さらに2週間が経ちました
zundamon: うぅ...頭が痛いのだ...
metan: ずんだもん、それ{food_name}の食べ過ぎじゃないですか？
zundamon: 違うのだ！{food_name}は関係ないのだ！
metan: でも、明らかに様子がおかしいですよ
zundamon: 仕事のストレスなのだ！
metan: 本当にそうですか...？
zundamon: そうなのだ！{food_name}のせいじゃないのだ！
metan: じゃあ、なんで毎日食べ始めてから調子悪いんですか
zundamon: それは...た、たまたまなのだ！
metan: もう病院に行きましょう
zundamon: まだ大丈夫なのだ...
```

## 重要な指示

- 症状の段階的悪化を丁寧に描写
- ずんだもんの心理変化を自然に表現
- 友人の心配が段階的に強まる様子
- 医学的に正確な症状（アウトラインと参考情報を活用）
- セリフ数は25-35の範囲で、自然な会話の流れを優先
