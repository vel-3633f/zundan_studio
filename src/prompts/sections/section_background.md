# セクション2: 食品解説・背景情報

## このセクションの役割

視聴者に{food_name}の基本情報を提供し、科学的な前提知識を与える。

## 生成内容

- **セリフ数**: 最低10セリフ、最大15セリフ
- **キャラクター構成**: narrator + metan（2人のみ）
- **背景**: library, classroom, laboratory など知的な背景

## 【重要】シーン導入文章

このセクションの最初のセリフは、ナレーターによるシーン導入文章です。
- visible_charactersには metan のみ（ナレーターは含めない）
- 例: "それでは、まず{food_name}について詳しく見ていきましょう"

## 具体的な内容要求

### ナレーターの解説内容（ですます調厳守）

1. **シーン導入**（1セリフ）
   - 「それでは、{food_name}について詳しく見ていきましょう」

2. **{food_name}の基本情報**
   - 「{food_name}は、〜〜な食品です」
   - 主要な栄養成分（参考情報を活用）

3. **一般的な健康効果**
   - 「適量であれば、〜〜という効果があります」
   - メリット部分を説明（参考情報のbenefits活用）

4. **適正摂取量の目安**
   - 「1日の摂取目安は〜〜です」

5. **橋渡しの一言**
   - 「しかし、毎日過剰に食べ続けるとどうなるのでしょうか？」

### めたんの役割

- ナレーターの説明への相槌や補足質問
- 「へぇ、そうなんですね」
- 「でも、食べ過ぎたらどうなるんですか？」
- 視聴者代表として自然な疑問を投げかける
- 表情: normal, thinking, worried

### 会話の流れ例

```
narrator: それでは、まず{food_name}について詳しく見ていきましょう
metan: お願いします
narrator: {food_name}は、〜〜な栄養素が含まれています
metan: へぇ、そうなんですね
narrator: 適量であれば、〜〜という効果があります
metan: 良いことも多いんですね
narrator: しかし、1日の摂取目安は〜〜です
metan: じゃあ、それ以上食べたらどうなるんですか？
narrator: では、その点について見ていきましょう...
```

## 重要な指示

- ナレーターは必ず「ですます調」で統一
- 参考情報の医学的・科学的根拠を活用
- めたんとナレーターの会話を交互に配置
- 視聴者が理解しやすい平易な言葉で
- 次のセクション（日常導入）への橋渡しを意識

## 出力形式

{format_instructions}

必ずVideoSectionオブジェクトとして出力:
- section_name: "食品解説・背景情報"
- scene_background: library, classroom, laboratory のいずれか
- segments: 10-15個（最初はナレーターのシーン導入から）
