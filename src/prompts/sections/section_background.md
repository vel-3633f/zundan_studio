# セクション 2: 食品解説・背景情報

## このセクションの役割

視聴者に{food_name}の基本情報を提供し、科学的な前提知識を与える。

## 生成内容

- **セリフ数**: 最低 10 セリフ、最大 15 セリフ
- **キャラクター構成**: narrator + metan（2 人のみ）
- **背景**: 教育的・知的な雰囲気を表す背景を動的に選択
  - 学習場所
  - 科学的な場所
  - 情報提示に適した場所
  - 解説内容に最も適した背景を選択

## 【重要】シーン導入文章

このセクションの最初のセリフは、ナレーターによるシーン導入文章です。

- visible_characters には metan のみ（ナレーターは含めない）
- 例: "それでは、まず{food_name}について詳しく見ていきましょう"

## このセクションの目的

視聴者に{food_name}の基本情報と科学的な前提知識を提供する

## 展開の流れ

1. シーン導入
2. {food_name}の基本情報の提示
3. 適正摂取の場合のメリット
4. 過剰摂取への懸念の提示
5. 次のセクションへの橋渡し

各段階で自然な会話を構成してください。

## 必須要素

- {food_name}の基本情報（参考情報を活用）
- 主要な栄養成分
- 適量の場合の健康効果（参考情報の benefits 活用）
- 適正摂取量の目安
- 過剰摂取のリスクへの言及
- めたんの相槌や質問で視聴者代表の役割（表情: normal, thinking, worried）

## 重要な指示

- ナレーターは必ず「ですます調」で統一
- 参考情報の医学的・科学的根拠を活用
- めたんとナレーターの会話を交互に配置
- 視聴者が理解しやすい平易な言葉で
- 次のセクション（日常導入）への橋渡しを意識

## アイテム画像の指定

**【重要】3-7 セリフごとに 1 つの教育アイテム画像を指定してください**

このセクションでは、ナレーターの解説を視覚的に補助するアイテム画像を表示できます。

### 基本ルール

- セリフの内容に応じた適切なアイテム ID を**動的に考えて**指定
- 目安として 5 セリフごとに画像を変更しますが、3-7 セリフの範囲で柔軟に調整可能
- **最初のセリフで必ずアイテムを指定**してください（セクション全体で表示され続けます）
- 画像を変更するセリフで `display_item` にアイテム ID を指定
- それ以外のセリフでは `display_item` を null または省略
- 画像が指定されたセリフ以降、次の画像が指定されるまで同じ画像が表示され続けます
- **アイテムは基本的に常に表示され続けます**（セクション開始から終了まで）

### アイテム ID の命名ルール

- 説明内容を端的に表す英語または英数字の識別子
- 小文字とアンダースコアを使用（例: `ramen`, `vitamin_b`, `sodium_chart`）
- {food_name}そのものを表示する場合は食品名をそのまま使用
- 具体的な名前を付ける

**注意**: アイテム ID は自由に設定できます。説明内容を最もよく表す ID を考えて指定してください。
