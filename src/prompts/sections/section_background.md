# セクション 2: 食品解説・背景情報

## セクション概要

- **セリフ数**: 10-15 セリフ
- **キャラクター**: narrator + metan
- **背景**: modern_study_room（固定）
- **役割**: 視聴者に{{food_name}}の基本情報と科学的な前提知識を提供

## 【重要】シーン導入文章

最初のセリフはナレーターによるシーン導入文章です。

- visible_characters には metan のみ（ナレーターは含めない）

## 必須要素

- {{food_name}}の基本情報（参考情報を活用）
- 主要な栄養成分
- 適量の場合の健康効果
- 適正摂取量の目安
- 過剰摂取のリスクへの言及
- めたんの相槌や質問で視聴者代表の役割（表情: normal, thinking, worried）

## 重要な指示

- ナレーターは必ず「ですます調」で統一
- **めたんはナレーターと話す時は敬語を使用**（「〜ですよ」「〜ですね」「〜ますか」など）
- 参考情報の医学的・科学的根拠を活用
- めたんとナレーターの会話を交互に配置
- 視聴者が理解しやすい平易な言葉で

- 最初のセリフで必ずアイテムを指定
- セリフの内容に応じた適切なアイテム ID を動的に考えて指定
- アイテム ID の命名ルールは共通ルールを参照（3 単語形式、実物のビジュアルまたは抽象概念）
