# セクション 3: 日常導入・理由付け

## セクション概要

- **セリフ数**: 12-18 セリフ
- **キャラクター**: zundamon + tsumugi
- **役割**: ずんだもんがなぜ{{food_name}}を毎日食べることになったのか、共感できる理由を提示

## 【重要】シーン導入文章

最初はナレーターのシーン導入（visible_characters に narrator は含めない）

## 【重要】一日 3 食の設定

- ずんだもんは**一日 3 食（朝・昼・夜）すべて**を{{food_name}}で済ませています
- この極端な食生活を会話の中で自然に明示してください

## 食べ物の特性を活かした理由付け（必須）

{{food_name}}の特性を考慮し、**その食べ物ならではの理由**を作ってください。一般的な理由ではなく、**{{food_name}}だからこそ**の理由を作ることを最優先してください。

参考情報の「メリット」情報を都合よく解釈させ、ずんだもんの楽観的な理由付けに使ってください。

## 必須要素

- アウトラインの「eating_reason」（{{outline_eating_reason}}）を反映しつつ、{{food_name}}の特性を強く反映
- **一日 3 食すべてを{{food_name}}にしている事実を会話内で明示**
- {{food_name}}の特性に応じた、現代人が共感できる「あるある」な理由
- ずんだもんの楽観的で前向きな態度（表情: happy, normal, excited）
- ずんだもんの性格バリエーション（データ信者型、流行追従型、専門家気取り型などが効果的）
- つむぎの素朴な質問と軽い心配（表情: worried, thinking, normal）
- つむぎの純粋な疑問がずんだもんの説明を引き出す

## 重要な指示

- ずんだもんの楽観的な性格を強調
- 現代人が共感できる「あるある」な理由
- つむぎの素朴な心配を一蹴するずんだもんの強気な態度
- つむぎの純粋な質問で視聴者代表の役割
- 次のセクション（楽観期）へスムーズに繋がる前向きな終わり方
- **視聴者への投げかけ**: セクション内で 1-2 回、視聴者に共感を求めるセリフを自然に入れる
