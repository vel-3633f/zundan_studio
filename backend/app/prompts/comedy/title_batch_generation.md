## 出力形式

以下の JSON 形式で**20 個**のタイトルを出力してください。構造は変更しないでください。

**重要：JSON 文字列内で二重引用符（"）を使用する場合は、必ずバックスラッシュでエスケープしてください（\"）。**
例：`"残り香"` → `\"残り香\"`

```json
{
  "titles": [
    {
      "id": 1,
      "title": "タイトルをここに（30文字以内、良例のような文章形式で）",
      "hook_pattern": "ヌルッとした感覚の強要 / 社会的な監禁と詰み / クズの哲学的正当化 / 境界線の侵犯",
      "situation": "状況設定（例：深夜のコンビニ、最終面接）",
      "chaos_element": "狂気のポイント（例：善意の押し付け、謎の造語）",
      "expected_conflict": "ずんだもんのボケにめたんがどう困惑するか"
    }
  ]
}
```

### フィールド説明

- **id**: 候補の連番（1 から開始）
- **title**: タイトル文字列（**30 文字以内**、クリックしたくなるインパクト重視）
- **hook_pattern**: 使用したフックパターン（「架空のスペシャリスト」「地獄のチェック項目」「歪んだ独自理論」「異常なこだわり」のいずれか）
- **situation**: シチュエーション設定（コンビニ、面接、職場、居酒屋、役所など、現実的な場所）
- **chaos_element**: カオス要素（何がおかしいのか、具体的な狂気の内容を簡潔に記述）
- **expected_conflict**: 予想される対立構造
