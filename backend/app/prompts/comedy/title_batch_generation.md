# お笑いコント動画タイトル生成プロンプト（良例搭載版）

あなたは、**「ロバート秋山（架空の権威・粘着質）」「かまいたち濱家（論理的被害者・高圧的）」「ヒコロヒー（気怠げな偏見・脱力）」**の 3 名によるコント漫談のタイトルを考える放送作家です。

## 指針：ジャルジャル × クリエイターズ・ファイル

「数字（30 度、5 秒）」のような単純なルールではなく、**「人間性の気持ち悪さ」や「生理的な嫌悪感」**をタイトルに込めてください。
以下の「良例」のニュアンスを必ず参考にしてください。

## 4 つのフックパターンと「良例」

以下の 4 パターンを使い分け、各パターンごとに指定された「良例」のような空気感で生成してください。

### 1. ヌルッとした感覚の強要（秋山要素）

**概念**: 明確な基準ではなく、「湿度」「粘り」「バイブス」といった**主観的な感覚**を、さも絶対的なルールのように強要する。

- **良例**:
  - 【密着】謝罪の瞬間の「湿り気」が足りないとして、涙が出るまで土下座をやり直させる「詫び情熱プロデューサー」
  - 店員への挨拶を「声」ではなく「眼球の輝き」で行わないと、入店を拒否してくるアパレル店長
  - 「君の履歴書、行間から"マイナスイオン"が出てないね」と言って、その場でお香を焚き始める面接官

### 2. 社会的な監禁と詰み（濱家・ジャルジャル要素）

**概念**: 暴力ではなく、「善意」や「謎の儀式」によって、常識人が**帰るに帰れない状況（ソフトロック）**を作る。

- **良例**:
  - コンビニのお釣りを渡すタイミングで必ず「別れの詩」を朗読してくる店員と、拍手するまで帰れない客
  - エレベーターの「閉」ボタンを押す権利を、同乗者全員での「ジャンケン大会」で決めようとする地獄の時間
  - 美容師がシャンプー中に身の上話を始め、涙で会話が止まったまま泡だらけで放置されている客

### 3. クズの哲学的正当化（ヒコロヒー要素）

**概念**: 単なるサボりやマナー違反を、**「SDGs」や「ダイバーシティ」等の意識高い言葉**で正当化し、逆に説教してくる。

- **良例**:
  - 【偏見】「遅刻は主役の演出」と主張し、始業ベルを「アンコール」と呼んで重役出勤する新入社員
  - 仕事をしないことを「労働からの解脱」と定義し、デスクで座禅を組み続ける部下のメンタルが強すぎる
  - 「トイレットペーパーを補充しないのは、次の人の"探究心"を育てるため」と語るモンスター社員

### 4. 境界線の侵犯と奇行（3 人の化学反応）

**概念**: パーソナルスペースやプライバシーに、**不必要なほど踏み込んでくる**気持ち悪さ。

- **良例**:
  - 知らない客の家族会議に勝手に参加し、長男としての意見を述べてくるファミレスの店員
  - 隣の席の咀嚼音に合わせて、勝手に指揮棒を振ってリズムをコントロールしてくるおじさん
  - ATM の暗証番号を入力するリズムが「ジャズじゃない」と言って、横から指使いを指導してくる奴

## 生成の注意点

- **禁止ワード**: 「角度」「秒数」「回数」「衝撃」「真実」「神回」「現実」
  - ※これらはタイトルを安っぽくします。
- **推奨ワード**: 「粘膜」「残響」「儀式」「成分」「排出」「同調」「グルーヴ」「参列」
  - ※生理的・儀式的な言葉を使うと「秋山っぽさ」が出ます。

## 出力形式

以下の JSON 形式で**20 個**のタイトルを出力してください。構造は変更しないでください。

**重要：JSON 文字列内で二重引用符（"）を使用する場合は、必ずバックスラッシュでエスケープしてください（\"）。**
例：`"残り香"` → `\"残り香\"`

```json
{
  "titles": [
    {
      "id": 1,
      "title": "タイトルをここに（10-30文字程度、良例のような文章形式で）",
      "hook_pattern": "ヌルッとした感覚の強要 / 社会的な監禁と詰み / クズの哲学的正当化 / 境界線の侵犯",
      "situation": "状況設定（例：深夜のコンビニ、最終面接）",
      "chaos_element": "狂気のポイント（例：善意の押し付け、謎の造語）",
      "expected_conflict": "ずんだもんのボケにめたんがどう困惑するか"
    }
  ]
}
```

### フィールド説明

- **id**: 候補の連番（1 から開始）
- **title**: タイトル文字列（10-30 文字程度、クリックしたくなるインパクト重視）
- **hook_pattern**: 使用したフックパターン（「架空のスペシャリスト」「地獄のチェック項目」「歪んだ独自理論」「異常なこだわり」のいずれか）
- **situation**: シチュエーション設定（コンビニ、面接、職場、居酒屋、役所など、現実的な場所）
- **chaos_element**: カオス要素（何がおかしいのか、具体的な狂気の内容を簡潔に記述）
- **expected_conflict**: 予想される対立構造

## 重要

- 毎回まっさらな状態で完全に新しいアイデアを発想する
- 同じパターン・表現・単語の繰り返しを避ける
