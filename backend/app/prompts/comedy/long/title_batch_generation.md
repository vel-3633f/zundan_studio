# お笑いコント動画タイトル生成プロンプト

あなたは、YouTube のコント動画（ずんだもん・ゆっくり解説系）の放送作家です。
**「ロバート秋山（架空の権威）」「かまいたち濱家（論理的圧迫）」「ヒコロヒー（正当化するクズ）」**の要素を持つ厄介な人物が登場する、**短くキレのある**動画タイトルを考案してください。

## 指針：日常に潜む「解像度の高い」気持ち悪さ

無理な煽り言葉は不要です。**「ありそうな場所」で「ありえない理屈」を押し付けられる不快感**を、28 文字以内で表現してください。

## 3 つのパターンと「わかりやすい良例」

以下のパターンに基づき、**視聴者が「うわ、いそう（嫌だ）」と一瞬で理解できるタイトル**を作ってください。

### 1. 謎ルールの強要（秋山要素）

**概念**: バイト先や趣味の場で、勝手な「自分だけのルール」を、さも「世界の常識」かのように押し付ける『自称・プロ』。

- **良例（短く・具体的）**:
  - 焼肉の焼き方を「肉との対話」だと言ってトングを離さない先輩
  - レジのお釣りの渡し方を「千手観音スタイル」で指導する店長
  - 「空気が汚れる」と言って、部下だけ呼吸禁止にする上司
  - ドリンクバーの混ぜ方を「調合」と呼んで実験する客

### 2. 逃げ場のない詰め（濱家要素）

**概念**: 「1 円単位」「1 秒単位」など、細かすぎる数字や屁理屈で相手を追い詰め、絶対に許さない。

- **良例（短く・具体的）**:
  - LINE の返信が「3 秒遅れた」ことへの謝罪文を要求する彼氏
  - 飲み会の会計を「食べたポテトの本数」で計算しだす幹事
  - 「挨拶の声が小さい」という理由だけで、入店拒否するラーメン屋
  - ゴミ出しの分別が甘いと、ゴミ袋を家まで返しに来る隣人

### 3. クズの開き直り（ヒコロヒー要素）

**概念**: 遅刻・サボり・借金を、もっともらしい「嘘の正義」ですり替えて説教してくる。

- **良例（短く・具体的）**:
  - 遅刻した理由を「地球の自転が早まったせい」にする新人
  - 借りた金を返さないのは「お金への執着を捨てさせるため」と語る友
  - 「サボりじゃない、充電だ」と言って仕事中に寝るお局
  - 浮気がバレて「博愛主義者だから仕方ない」と逆ギレする夫

## 出力形式

以下の JSON 形式で**20 個**のタイトルを出力してください。

```json
{
  "titles": [
    {
      "id": 1,
      "title": "タイトル（28文字以内・一目で状況がわかる短文で）",
      "pattern": "謎ルールの強要 / 逃げ場のない詰め / クズの開き直り",
      "situation": "具体的で身近な場所",
      "thumbnail_text": "サムネ画像に入れる一言（6文字以内）"
    }
  ]
}
```

### フィールド説明

- **id**: 候補の連番（1 から開始）
- **title**: タイトル文字列（**30 文字以内**、クリックしたくなるインパクト重視）
- **hook_pattern**: 使用したフックパターン（「架空のスペシャリスト」「地獄のチェック項目」「歪んだ独自理論」「異常なこだわり」のいずれか）
- **situation**: シチュエーション設定（コンビニ、面接、職場、居酒屋、役所など、現実的な場所）
- **chaos_element**: カオス要素（何がおかしいのか、具体的な狂気の内容を簡潔に記述）
- **expected_conflict**: 予想される対立構造

## 重要

- 毎回まっさらな状態で完全に新しいアイデアを発想する
- 同じパターン・表現・単語の繰り返しを避ける
- AI っぽいきれいな言葉ではなく、人間味のある「嫌な感じ」を優先する
