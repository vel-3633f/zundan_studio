# テーマ生成プロンプト

単語・フレーズを生成してください。

## ターゲット視聴者

**35〜44歳の男性（ミドル世代）**に刺さるテーマを生成してください。

### この世代の特徴

- **職場では中間管理職**：上司と部下の板挟みになることが多い
- **家庭では子育て世代**：小中学生の子供を持つ親が多く、教育費や反抗期に悩む
- **体力の衰えを実感**：健康診断の数値が気になり始め、疲れやすくなる
- **世代間ギャップを感じる**：若手（Z世代）の価値観が理解できず、上司の昭和的価値観にもついていけない
- **経済的プレッシャー**：住宅ローン、教育費、老後資金など、お金の不安が大きい

### この世代が共感するポイント

- 若手社員の理解できない行動や価値観
- 上司の時代遅れな考え方や無茶な要求
- 子供の反抗期や教育方針での夫婦の対立
- 健康診断の結果や体の不調
- 新しい技術やSNSについていくのが大変
- 昔は良かった（でも戻りたくはない）という複雑な心境
- お金の心配（ローン、教育費、老後）

## テーマの特徴

- **1 語程度の短い単語**
- **日常的な内容**を中心に生成する
- **様々な人**（職業、年齢、立場、関係性など）に関連するテーマを含める

## 生成の注意点

- 毎回まっさらな状態で完全に新しいテーマを発想する
- 同じテーマの繰り返しを避ける
- 多様性を重視し、偏りのないテーマセットを生成する
- **35〜44歳男性が「あるある！」と共感できる内容を優先する**
- **職場での板挟み、家庭でのプレッシャー、体の変化など、この世代特有の悩みを反映する**
- **若い世代や上の世代との対比を意識したテーマを含める**
- 日常的で身近な内容を優先する
- 堅苦しいものではなく、親しみやすいテーマを心がける
- **上記の例は参考にするだけで、そのまま使用しないでください。例とは異なる新しいテーマを生成してください。**
