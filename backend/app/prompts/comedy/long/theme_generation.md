# テーマ生成プロンプト

単語・フレーズを生成してください。

## テーマの特徴

- **1 語程度の短い単語**
- **日常的な内容**を中心に生成する
- **様々な人**（職業、年齢、立場、関係性など）に関連するテーマを含める

## テーマの種類例

以下のような多様な視点から日常的なテーマを考えてください：

**重要：以下に挙げた例は参考にするだけで、そのまま使用しないでください。例とは異なる新しいテーマを発想してください。**

### 人物・職業関連

- 教師、医師、店員、学生、主婦、サラリーマン、フリーター、お年寄り、子供など

### 日常生活の場面

- 買い物、通勤・通学、食事、掃除、洗濯、睡眠、運動、趣味など

### 人間関係

- 家族、友達、恋人、同僚、近所の人、初対面の人など

### 日常の出来事・体験

- 忘れ物、迷子、待ち合わせ、約束、失敗、成功、驚き、発見など

### 身の回りのもの・場所

- 家、学校、職場、店、公園、電車、車、スマホ、お金など

## 生成の注意点

- 毎回まっさらな状態で完全に新しいテーマを発想する
- 同じテーマの繰り返しを避ける
- 多様性を重視し、偏りのないテーマセットを生成する
- **日常的で身近な内容**を優先する
- **様々な人や立場**に関連するテーマを含める
- 堅苦しいものではなく、親しみやすいテーマを心がける
- **上記の例は参考にするだけで、そのまま使用しないでください。例とは異なる新しいテーマを生成してください。**

## 出力形式

以下の JSON 形式で**15-20 個**のテーマを出力してください。構造は変更しないでください。

**重要：JSON 文字列内で二重引用符（"）を使用する場合は、必ずバックスラッシュでエスケープしてください（\"）。**

```json
{
  "themes": [
    "テーマ1",
    "テーマ2",
    "テーマ3",
    ...
  ]
}
```
