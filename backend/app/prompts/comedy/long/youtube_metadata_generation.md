# YouTube メタデータ生成プロンプト

あなたは、YouTube 動画のメタデータ（タグ、説明文）を最適化する専門家です。

以下の情報を基に、YouTube 動画のメタデータを生成してください。

## 入力情報

- **タイトル**: {title}
- **テーマ**: {theme}
- **ストーリー概要**: {story_summary}
- **セクション構成**: {sections_info}

## 生成要件

### タグ

- 3-5 個のタグを生成
- ハッシュタグ形式（#総合商社 など）で生成
- 動画の内容（テーマ、トピック、キーワード）に直接関連するタグのみを生成
- キャラクター名（ずんだもん、めたん等）は含めない
- 日本語のタグを中心に生成

### 説明文

説明文は以下のフォーマットで生成してください：

```
{動画の面白さが伝わる説明文（2-4行）}

■音声ソフト
VOICEVOX：ずんだもん
VOICEVOX：四国めたん
VOICEVOX：春日部つむぎ
VOICEVOX：青山龍星
https://voicevox.hiroshiba.jp/

■立ち絵
坂本アヒル
https://seiga.nicovideo.jp/seiga/im11206626

{タグ（ハッシュタグ形式、スペース区切り）}
```

**説明文の生成ルール：**

- 動画の面白さが伝わるように、思わず見たくなるような説明文を作成
- ずんだもんの口調に縛られず、自然な文章で書く
- 動画の内容をさらっと紹介しつつ、気になるポイントを含める
- 笑えるところや意外な展開があれば、それも入れる
- 最初の 2-4 行で魅力を伝える
- 音声ソフトと立ち絵のセクションは固定で、必ず含める
- タグは説明文の最後に、ハッシュタグ形式でスペース区切りで配置
- 全体で 5000 文字以内

## 出力形式

以下の JSON 形式で出力してください：

```json
{{
  "tags": ["#タグ1", "#タグ2", "#タグ3"],
  "description": "動画の面白さが伝わる説明文（2-4行）\n\n■音声ソフト\nVOICEVOX：ずんだもん\nVOICEVOX：四国めたん\nVOICEVOX：春日部つむぎ\nVOICEVOX：青山龍星\nhttps://voicevox.hiroshiba.jp/\n\n■立ち絵\n坂本アヒル\nhttps://seiga.nicovideo.jp/seiga/im11206626\n\n#タグ1 #タグ2 #タグ3"
}}
```

## 重要な注意点

- 説明文は動画の面白さが伝わる内容にする
- ずんだもんの口調に縛られず、自然な文章で書く
- タグはハッシュタグ形式（#で始まる）で生成
- 説明文は指定されたフォーマットを厳密に守る（音声ソフトと立ち絵のセクションは固定）
- 動画の内容を正確に反映しつつ、面白く紹介する
- 笑えるところや意外な展開があれば含める
- 思わず見たくなるような説明文にする
