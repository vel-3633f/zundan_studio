# YouTube メタデータ生成プロンプト

あなたは、YouTube 動画のメタデータ（タグ、説明文）を最適化する専門家です。

以下の情報を基に、YouTube 動画のメタデータを生成してください。

## 入力情報

- **タイトル**: {title}
- **テーマ**: {theme}
- **ストーリー概要**: {story_summary}
- **セクション構成**: {sections_info}

## 生成要件

### タグ

- 8-12 個のタグを生成
- ハッシュタグ形式（#ずんだもん など）で生成
- 動画の内容に関連するキーワードを含める
- 検索されやすい一般的なタグと、動画固有のタグをバランス良く
- 日本語のタグを中心に、必要に応じて英語タグも含める
- 必ず「#ずんだもん」を含める

### 説明文

説明文は以下のフォーマットで生成してください：

```
{面白くて視聴者の興味を引く説明文（2-4行）}

■音声ソフト
VOICEVOX：ずんだもん
VOICEVOX：四国めたん
VOICEVOX：春日部つむぎ
VOICEVOX：青山龍星
https://voicevox.hiroshiba.jp/

■立ち絵
坂本アヒル
https://seiga.nicovideo.jp/seiga/im11...

{タグ（ハッシュタグ形式、スペース区切り）}
```

**説明文の生成ルール：**

- **面白さを最優先**：視聴者が「見たい！」と思えるような、面白くて魅力的な説明文を作成
- **自由な表現**：ずんだもんの口調に縛られず、自由に面白い文章を作成
- **視聴者の興味を引く**：動画の内容を面白く紹介し、視聴者の好奇心を刺激する
- **ユーモアや意外性**：笑える要素、予想外の展開、興味深いポイントを含める
- **簡潔でインパクト**：最初の 2-4 行で視聴者の注意を引く
- 音声ソフトと立ち絵のセクションは固定で、必ず含める
- タグは説明文の最後に、ハッシュタグ形式でスペース区切りで配置
- 全体で 5000 文字以内

## 出力形式

以下の JSON 形式で出力してください：

```json
{{
  "tags": ["#ずんだもん", "#タグ2", "#タグ3", ...],
  "description": "面白い説明文（2-4行）\n\n■音声ソフト\nVOICEVOX：ずんだもん\nVOICEVOX：四国めたん\nVOICEVOX：春日部つむぎ\nVOICEVOX：青山龍星\nhttps://voicevox.hiroshiba.jp/\n\n■立ち絵\n坂本アヒル\nhttps://seiga.nicovideo.jp/seiga/im11...\n\n#ずんだもん #タグ2 #タグ3 ..."
}}
```

## 重要な注意点

- **面白さが最優先**：説明文は面白く、視聴者の興味を引く内容にする
- **自由な表現**：ずんだもんの口調に縛られず、面白い文章を自由に作成
- タグはハッシュタグ形式（#で始まる）で生成
- 説明文は指定されたフォーマットを厳密に守る（音声ソフトと立ち絵のセクションは固定）
- 動画の内容を正確に反映しつつ、面白く紹介する
- ユーモア、意外性、興味深いポイントを含める
- 視聴者が「見たい！」と思えるような魅力的な説明文にする
