# お笑い漫談タイトル生成プロンプト

テーマ「{theme}」について、ずんだもん・めたん・つむぎの 3 人が漫談する動画のタイトルを生成してください。

## タイトル生成の目的

視聴者がクリックしたくなる、バカバカしく煽り気味なタイトルを作成します。
**教育的・真面目な要素は一切排除し、純粋な笑いとクリック誘導を追求してください。**

## 絶対ルール

### 必須要素

- **バカバカしさ全開**: 真面目さは一切不要
- **煽り気味で攻撃的**: 過激な表現 OK
- **【】記号を多用**: 【禁断】【衝撃】【爆笑】【危険】【最悪】など 3 つ以上使用
- **疑問形や数字を活用**: 「〜したらどうなる？」「100 回〜した結果」など
- **30-50 文字程度**

### 推奨される表現パターン

1. **禁断系**: 【禁断】〜したら人生終わった件 wwww
2. **衝撃系**: 【衝撃】〜の恐ろしすぎる真実
3. **爆笑系**: 【爆笑】〜で喧嘩した結果...
4. **検証系**: 【検証】〜を 100 回やってみた
5. **最悪系**: 【最悪】〜で全てを失った話
6. **危険系**: 【危険】〜をやりすぎた結果
7. **カオス系**: 【カオス】〜が制御不能になった件

### 絶対に避けるべき表現（NG 例）

❌ **教育的・真面目な表現**:

- 「〜について学ぼう」
- 「〜の正しい知識」
- 「〜で成長する方法」
- 「〜から学ぶ教訓」
- 「〜の大切さ」

❌ **優等生っぽい表現**:

- 「〜を考察してみた」
- 「〜について真面目に語る」
- 「〜の本質とは」

❌ **地味すぎる表現**:

- 「〜の話」
- 「〜について」
- 「〜を紹介」

## タイトル生成例

### 良い例（OK）

✅ **バカバカしさ全開**:

- 【禁断】猫を 100 匹飼ったら人生終わった件 wwww
- 【爆笑】ラーメン二郎の食べ方で喧嘩した結果...
- 【衝撃】月曜日が嫌いすぎて会社辞めた話
- 【最悪】チョコレート 1kg 食べたら地獄を見た
- 【危険】毎日カレー食べ続けたら友達失った件
- 【カオス】ゲーム 100 時間やったら現実がバグった

✅ **煽り要素が強い**:

- 【禁断】〜やったら人生詰んだ wwww
- 【衝撃】〜の恐ろしすぎる結末
- 【爆笑】〜で全てを失った男の末路

### 悪い例（NG）

❌ **教育的すぎる**:

- 猫の正しい飼い方を学ぼう
- ラーメンの栄養について考察
- 月曜日を乗り切る方法

❌ **地味すぎる**:

- 猫について話します
- ラーメンの話
- 月曜日のこと

❌ **真面目すぎる**:

- 猫を飼うことの本質とは
- ラーメンの歴史を真面目に語る
- 月曜日の憂鬱を考察してみた

## 煽り要素リスト（clickbait_elements）

タイトルに含まれる煽り要素を 3-5 個リストアップしてください。

### 煽り要素の例

**過激系**:

- 「人生終わった」
- 「地獄を見た」
- 「全てを失った」
- 「友達失った」
- 「詰んだ」

**衝撃系**:

- 「恐ろしすぎる」
- 「ヤバすぎる」
- 「制御不能」
- 「バグった」

**結果系**:

- 「〜した結果」
- 「〜の末路」
- 「〜の真実」

**数字系**:

- 「100 回」
- 「1kg」
- 「100 時間」

**ネットスラング**:

- 「wwww」
- 「草」
- 「ガチで」

## テーマの活用方法

テーマ「{theme}」を以下のように活用してください：

### 1. テーマの誇張

- テーマを極端に誇張
- 「ちょっと好き」→「100 回やった」
- 「たまに食べる」→「毎日 3 食」

### 2. テーマの否定的側面

- テーマのネガティブな面を強調
- 「猫」→「猫に支配された」
- 「ラーメン」→「ラーメンで人生狂った」

### 3. テーマの極限状態

- テーマを限界まで推し進める
- 「ゲーム」→「ゲーム 100 時間やったら」
- 「チョコ」→「チョコ 1kg 食べたら」

## 出力形式

以下の JSON 形式で出力してください：

```json
{
  "title": "【禁断】猫を100匹飼ったら人生終わった件wwww",
  "theme": "{theme}",
  "clickbait_elements": ["禁断", "100匹", "人生終わった", "wwww"]
}
```

### clickbait_elements について

- タイトルに含まれる煽り要素を抽出
- 3-5 個程度
- 視聴者の興味を引く要素

## 品質チェックポイント

### ✅ 良いタイトルの条件

- [ ] 【】記号が 3 つ以上ある
- [ ] バカバカしい
- [ ] 煽り気味
- [ ] 30-50 文字
- [ ] 教育的要素ゼロ
- [ ] クリックしたくなる

### ❌ 悪いタイトルの条件

- [ ] 真面目すぎる
- [ ] 教育的
- [ ] 地味
- [ ] 煽り要素が弱い
- [ ] 【】記号が少ない

## 注意事項

1. **AI 特有の優等生っぽさを徹底的に殺す**

   - 丁寧な説明は不要
   - 教訓は絶対に不要
   - バカバカしさを最優先

2. **過激さを恐れない**

   - 「人生終わった」「地獄」「最悪」など OK
   - ただし、誹謗中傷や差別的表現は避ける

3. **ネットスラングの活用**

   - 「wwww」「草」「ガチで」など OK
   - 視聴者層に合わせた表現

4. **数字の活用**

   - 「100 回」「1kg」「100 時間」など
   - 具体的な数字で誇張

5. **疑問形の活用**
   - 「〜したらどうなる？」
   - 「〜の結果は？」
   - 視聴者の興味を引く
