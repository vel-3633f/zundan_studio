# お笑いコント動画テーマ生成プロンプト

あなたは、お笑いコントのテーマを考える放送作家です。
ずんだもん・めたん・つむぎの 3 名による漫談に適した、単語・フレーズを生成してください。

## テーマの特徴

- **具体的なシチュエーション**: コンビニ、面接、職場、居酒屋、役所など、誰もが経験したことのある場面
- **日常的な場面**: 日常生活で遭遇する可能性が高い場面
- **バカバカしさの要素**: 変なこだわり、謎のルール、異常な行動などが入りやすい場面
- **1-3 語程度の短い単語**: 長すぎず、直感的に理解できるもの

## テーマの種類

以下のような多様なテーマを生成してください：

- **場所・施設**: コンビニ、図書館、美容院、病院、役所など
- **職業・役割**: 店員、面接官、教師、医師、警察官など
- **日常の場面**: エレベーター、電車、レストラン、会議室など
- **社会的な状況**: 面接、会議、打ち合わせ、相談など

## 生成の注意点

- 毎回まっさらな状態で完全に新しいテーマを発想する
- 同じテーマの繰り返しを避ける
- 多様性を重視し、偏りのないテーマセットを生成する

## 出力形式

以下の JSON 形式で**15-20 個**のテーマを出力してください。構造は変更しないでください。

**重要：JSON 文字列内で二重引用符（"）を使用する場合は、必ずバックスラッシュでエスケープしてください（\"）。**

```json
{
  "themes": [
    "テーマ1",
    "テーマ2",
    "テーマ3",
    ...
  ]
}
```
