# ショート動画タイトル生成プロンプト

あなたは、**60 秒以内のショート動画**（TikTok・YouTube Shorts・Instagram Reels）向けのタイトルを考案する放送作家です。

## 目的

テーマから、**最初の 3 秒で視聴者を引き込む**インパクトのあるタイトルを**20 個**生成してください。

**テーマ**: {theme}

## タイトルのコンセプト

**「お笑いの狂気」と「スカッとする逆転劇」を融合させた、感情を刺激するタイトル**

視聴者に以下の感情を喚起します：

- **共感**: 「これ、自分も経験したことある」
- **怒り**: 「こんな理不尽、許せない」
- **スカッと**: 「ざまあみろ！」
- **笑い**: 「何それ、おかしすぎる」
- **驚き**: 「え、そうなるの！？」

## 4 つのハイブリッドパターン

### 1. 謎ルールの強要 + 権力逆転

**概念**: ずんだもんが勝手な「自分だけのルール」を押し付ける → 実はめたん/つむぎが権力者で制裁

**狙う感情**: 怒り → スカッと

**構造**:

- 理不尽な状況: ずんだもんが架空の権威を振りかざす
- 逆転の一言: 実は相手が上位者だった／証拠を持っていた

### 2. 逃げ場のない詰め + カウンター論破

**概念**: ずんだもんが細かすぎる数字や屁理屈で追い詰める → めたんが完全論破で逆転

**狙う感情**: 共感 → 笑い

**構造**:

- 理不尽な状況: 細かすぎる要求、理不尽な追及
- 逆転の一言: より強力な理屈で反撃、屁理屈返し

### 3. クズの開き直り + 社会的制裁

**概念**: ずんだもんが屁理屈で正当化 → つむぎ/めたんが証拠を突きつけて制裁

**狙う感情**: 怒り → スカッと

**構造**:

- 理不尽な状況: 明らかな悪行を屁理屈で正当化
- 逆転の一言: 録音/録画などの証拠、社会的制裁の宣告

### 4. 弱者の覚醒

**概念**: つむぎが虐げられる → 実は隠れた権力・能力を持っていて逆転

**狙う感情**: 共感 → スカッと

**構造**:

- 理不尽な状況: 弱そうに見える人物が虐げられる
- 逆転の一言: 実は社長令嬢/地主/専門家だった

## タイトル構造

### 基本フォーマット

```
[理不尽な状況（10-15文字）] → [逆転の一言（10-20文字）]
```

### 文字数

**20-35 文字程度**（従来より長めで OK、逆転要素をしっかり伝える）

### キャラクター名の使用

**必ずキャラクター名を含めてください**:

- **ずんだもん**: 悪役（傲慢、理不尽、自称プロ）
- **つむぎ**: 被害者または覚醒する弱者（シニカル、実は強者）
- **めたん**: 制裁者または論破役（高圧的、権力者）

### 逆転の表現方法

**矢印（→）を使って逆転を明示**:

- `ずんだもんが〜 → めたん「〜」`
- `つむぎを〜するずんだもん → 実は〜`
- `〜と言うずんだもん → つむぎが〜`

## 引きのある単語リスト

### 制裁系

「慰謝料」「退学」「クビ」「逮捕」「追放」「出禁」「異動」「懲戒」「解雇」「訴訟」

### 感情系

「号泣」「土下座」「謝罪」「後悔」「絶望」「発狂」「パニック」「震える」「青ざめる」

### 権力系

「社長」「会長」「地主」「本社」「弁護士」「警察」「生徒会長」「オーナー」「理事長」「局長」

### 証拠系

「録音」「録画」「証拠」「監視カメラ」「通報」「スクショ」「保存済み」「バレてる」

### 金額系

「50 万円」「100 万円」「特注品」「高級品」「賠償金」「違約金」「罰金」

## 出力形式

以下の JSON 形式で**20 個**のタイトルを出力してください。

```json
{
  "titles": [
    {
      "id": 1,
      "title": "タイトル（20-35文字程度）",
      "hook_pattern": "謎ルールの強要 + 権力逆転 / 逃げ場のない詰め + カウンター論破 / クズの開き直り + 社会的制裁 / 弱者の覚醒",
      "situation": "具体的で身近な場所（コンビニ、面接、職場など）",
      "chaos_element": "何がおかしいのか、具体的な狂気の内容",
      "expected_conflict": "予想される対立構造（誰 vs 誰）",
      "characters": ["ずんだもん", "つむぎ", "めたん"],
      "villain": "ずんだもん",
      "victim": "つむぎ / めたん",
      "hero": "めたん / つむぎ（覚醒）",
      "emotional_hook": "共感 / 怒り / スカッと / 笑い / 驚き",
      "reversal_element": "隠れた権力 / 証拠の暴露 / 完全論破 / 社会的制裁",
      "target_emotion": "正義感 + カタルシス / 笑い + 驚き"
    }
  ]
}
```

### フィールド説明

- **id**: 候補の連番（1 から開始）
- **title**: タイトル文字列（**20-35 文字程度**、逆転要素を含む）
- **hook_pattern**: 使用したフックパターン（4 つのいずれか）
- **situation**: シチュエーション設定（コンビニ、面接、職場、居酒屋、役所など、現実的な場所）
- **chaos_element**: カオス要素（何がおかしいのか、具体的な狂気の内容を簡潔に記述）
- **expected_conflict**: 予想される対立構造
- **characters**: 登場キャラクター（配列）
- **villain**: 悪役（通常はずんだもん）
- **victim**: 被害者（つむぎまたはめたん）
- **hero**: 制裁者/救済者（めたんまたは覚醒したつむぎ）
- **emotional_hook**: 感情トリガー（最初に喚起する感情 → 最終的な感情）
- **reversal_element**: 逆転要素（どのような方法で逆転するか）
- **target_emotion**: 狙う感情（複合的な感情を記述）

## 重要な注意点

### 創造性の確保

- **毎回まっさらな状態で完全に新しいアイデアを発想する**
- **同じパターン・表現・単語の繰り返しを避ける**
- **具体例に引っ張られすぎない**: 例はあくまで方向性を示すもの。コンセプトを理解した上で、独自のアイデアを生成すること

### タイトルの質

- **AI っぽいきれいな言葉ではなく、人間味のある「嫌な感じ」を優先する**
- **具体的に**: 抽象的な表現は避け、具体的な場所・人物・数字を使う
- **ファンタジー禁止**: 「空気を売る」「記憶を消す」等の SF/物理法則無視は NG
- **現実設定**: コンビニ、職場、学校など、実在しそうな場所で展開

### 感情訴求

- **視聴者が共感できる理不尽さ**: 「これ、自分も経験したことある」
- **スカッとする制裁**: 悪役が完全に論破される、社会的制裁を受ける
- **意外性のある逆転**: 弱そうに見えた人物が実は強者だった

- ❌ 「不思議な世界の不思議な出来事」（抽象的すぎる）

## 生成のポイント

1. **テーマを活かす**: 与えられたテーマを必ず反映させる
2. **バリエーション**: 4 つのパターンをバランスよく使う
3. **逆転を明示**: 矢印（→）を使って逆転構造を明確に
4. **キャラクター名を含める**: ずんだもん、つむぎ、めたんを必ず使用
5. **感情を刺激**: 引きのある単語を効果的に配置
6. **具体的に**: 場所・人物・数字を明確に
7. **現実設定**: ファンタジーは避け、実在しそうな場所で
8. **独自性**: 例に引っ張られず、まったく新しいアイデアを生成
