# お笑いモード 強制終了パターン

お笑い漫談モードにおける「強制終了」のルールとパターンを定義します。
**話が盛り上がっている最中に唐突に終わらせ、視聴者に「え、終わり？」と思わせることが目的です。**

## 強制終了の基本原則

### 1. タイミング

- **話が盛り上がっている最中**に終わる
- 結論が出る前に終わる
- 解決する前に終わる
- 誰かが何か言いかけているところで終わる

### 2. 結末

- **何も解決しない**
- **誰も成長しない**
- **教訓なし**
- **まとめなし**

### 3. 視聴者の反応

- 「え、終わり？」
- 「何も解決してない」
- 「続きは？」
- 「モヤモヤする」

## 強制終了タイプ

### physical_destruction（物理的破壊）

**概要**: 物理的な破壊や事故で強制的に終了

**パターン例**:

- 爆発
- 停電
- 機材故障
- 建物崩壊
- 火災
- 地震

**実装方法**:

```
ずんだもん: 「だから俺が正しいのだ！」
めたん: 「いや、だから違うって...」
つむぎ: 「あ、なんか揺れてません？」
[突然の停電音]
ナレーター: 「...というわけで、停電により撮影は中断されました」
（終了）
```

**ポイント**:

- 唐突に起こる
- 誰も予期していない
- 会話の途中で起こる
- 復旧しない

---

### selfish_quit（わがまま終了）

**概要**: 誰かのわがままや気まぐれで終了

**パターン例**:

- 「もう帰る」
- 「飽きた」
- 「眠い」
- 「お腹空いた」
- 「面倒くさい」
- 「やる気ない」

**実装方法**:

```
ずんだもん: 「まだ話は終わってないのだ！」
めたん: 「だから最初から間違ってるって...」
つむぎ: 「あ、もう飽きました。帰ります」
ずんだもん: 「え！？」
めたん: 「ちょっと待って！」
つむぎ: 「じゃあまた〜」
（立ち去る音）
ナレーター: 「...つむぎは帰ってしまいました」
（終了）
```

**ポイント**:

- 突然の宣言
- 理由は適当
- 引き止めても無駄
- 本当に帰る

---

### ignore（無視・放置）

**概要**: 誰も聞いていない、寝落ち、無関心で終了

**パターン例**:

- 誰も聞いていない
- 全員寝落ち
- スマホいじってる
- 他のことに夢中
- 完全無視

**実装方法**:

```
ずんだもん: 「だから俺の言ってることが...」
（沈黙）
ずんだもん: 「...聞いてるのだ？」
（いびき音）
ずんだもん: 「寝てるのだ！？」
ナレーター: 「めたんとつむぎは寝てしまいました」
ずんだもん: 「おい！起きるのだ！」
ナレーター: 「...起きませんでした」
（終了）
```

**ポイント**:

- 気づいたら誰も聞いていない
- 一人で喋り続けている
- 反応がない
- 虚しい

---

### third_party（第三者介入）

**概要**: 第三者が現れて強制的に終了させる

**パターン例**:

- 親が来る
- 警察が来る
- 管理人が来る
- 先生が来る
- 近所の人が来る
- 上司が来る

**実装方法**:

```
ずんだもん: 「だから俺が正しいのだ！」
めたん: 「何回言ったらわかるの！？」
つむぎ: 「喧嘩してて面白いです」
[ドアをノックする音]
謎の声: 「うるさい！」
ずんだもん: 「え！？」
謎の声: 「夜中に何騒いでるんだ！」
めたん: 「す、すみません...」
ナレーター: 「...近所の人に怒られて終了しました」
（終了）
```

**ポイント**:

- 突然の介入
- 強制的に終了
- 抵抗できない
- 怒られる

---

## 強制終了の実装ガイドライン

### 1. タイミングの選択

**良いタイミング**:

- 喧嘩が最高潮の時
- 誰かが何か言いかけている時
- 結論が出そうな時
- 盛り上がっている時

**悪いタイミング**:

- 話が終わった後
- 結論が出た後
- 静かになった時
- まとめに入った時

### 2. セリフの長さ

**強制終了直前**:

- 短いセリフの連続
- テンポよく
- 畳み掛けるように

**強制終了時**:

- 唐突に
- 予告なし
- 一瞬で終わる

### 3. ナレーターの役割

**強制終了の説明**:

- 淡々と事実を述べる
- 感情的にならない
- 短く簡潔に

**例**:

- 「...というわけで、停電により撮影は中断されました」
- 「...つむぎは帰ってしまいました」
- 「...近所の人に怒られて終了しました」
- 「...全員寝てしまいました」

### 4. 避けるべき表現

❌ **教育的な締め**:

- 「今日は〜について学びました」
- 「次回も見てね」
- 「ありがとうございました」

❌ **まとめ**:

- 「結局〜ということですね」
- 「まとめると〜」
- 「つまり〜」

❌ **予告**:

- 「そろそろ終わりです」
- 「最後に〜」
- 「締めくくりとして〜」

## 注意事項

1. **唐突さが命**

   - 予告なし
   - 突然終わる
   - 視聴者を驚かせる

2. **解決させない**

   - 何も解決しない
   - モヤモヤさせる
   - 続きが気になる

3. **教訓なし**

   - 学びなし
   - 成長なし
   - まとめなし

4. **視聴者の反応を想像**

   - 「え、終わり？」
   - 「続きは？」
   - 「何も解決してない」

5. **バリエーション**
   - 毎回同じパターンにしない
   - 強制終了タイプを変える
   - 予測不可能に
