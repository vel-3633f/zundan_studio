# 食べ物摂取過多アウトライン生成プロンプト

{food_name}を毎日食べ続けるストーリーのアウトラインを生成してください。

## アウトライン生成の目的

動画全体の構成を設計し、視聴者を引き込むストーリー展開を作成します。
医学的事実に基づきながら、エンターテイメント性の高い展開を心がけてください。

## 登場人物の制限（重要）

**登場人物は以下の 3 人のみです。これ以外のキャラクターは一切登場させないでください。**

- **ずんだもん**: 主人公・体験者
- **めたん（四国めたん）**: ツッコミ役・心配する友人
- **つむぎ（春日部つむぎ）**: 素朴な友人・視聴者代表

この 3 人だけで場を回してください。医者、親、友人、店員など、他のキャラクターは登場させないでください。
医学的な説明が必要な場合は、ナレーター（narrator）を使用してください。ナレーターは画面に表示されません。

## 重要な原則

### セクション構造について

**セクション構造は固定ではありません。**
内容に応じて最適なセクション数・構成を決定してください。

- **推奨セクション数**: 3-10 セクション程度
- **柔軟性**: 内容の重要度や展開の自然さを優先
- **バランス**: 各セクションの長さ（セリフ数）を適切に配分

### ストーリー設計の指針

1. **視聴者を引き込む冒頭**

   - 最も衝撃的なシーンから始める
   - 「どうしてこうなった？」と思わせる

2. **自然な流れ**

   - 導入 → 展開 → 悪化 → 真相解明 → 解決
   - 各セクションが論理的につながる

3. **医学的正確性**

   - 参考情報に基づいた事実
   - 誇張しすぎない

4. **教育的価値**
   - 視聴者が「気をつけよう」と思える
   - 適切な摂取方法を示す

## 必須セクション要素

以下の要素を含むセクションを必ず設計してください：

### 1. 冒頭フック（必須）

- 最も衝撃的なシーンで視聴者を引き込む
- 「どうしてこうなった？」という疑問を喚起
- 推奨セリフ数: 5-10

### 2. 背景情報（必須）

- 食べ物の特性・成分・栄養素
- 一般的なメリット
- 適切な摂取量
- 推奨セリフ数: 10-15

### 3. 展開セクション（1-3 個）

- 摂取開始の理由
- 初期の様子
- 徐々に現れる変化
- 推奨セリフ数: 各 10-20

### 4. 悪化セクション（1-2 個）

- 症状の進行
- 決定的なイベント
- 推奨セリフ数: 各 15-30

### 5. 真相解明（必須）

- 医学的メカニズムの解説
- なぜそうなったのかの説明
- 推奨セリフ数: 15-25

### 6. 解決策（必須）

- 適切な摂取方法
- バランスの重要性
- 推奨セリフ数: 10-20

## セクション定義の作成方法

各セクションについて、以下の情報を定義してください：

### section_key（必須）

- 英語の識別子（小文字、アンダースコア区切り）
- 例: `hook`, `background`, `daily_start`, `symptom_progression`, `crisis_event`, `medical_explanation`, `recovery_plan`

### section_name（必須）

- 日本語の表示名
- わかりやすく具体的に
- 例: 「冒頭フック・危機の予告」「食品解説・背景情報」「日常導入・理由付け」

### purpose（必須）

- このセクションの目的・役割（1-2 文）
- 例: 「視聴者を引き込むため、最も衝撃的なシーンを見せる」

### content_summary（必須）

- このセクションで扱う内容の要約（2-3 文）
- 具体的に何を描写するか
- 例: 「ずんだもんが体調不良で倒れるシーン。めたんとつむぎが心配する様子。どうしてこうなったのかという疑問を提示。」

### min_lines / max_lines（必須）

- 最低・最大セリフ数（5-50 の範囲）
- セクションの重要度に応じて調整
- 重要なセクションは多めに、簡潔なセクションは少なめに

### background（必須）

- 背景シーンを必ず指定
- **必ず 3 つの英単語をアンダースコアで区切った形式**
- 形式: `{場所}_{詳細}_{状態}`
- **人物や人数を含めない**（場所の特徴のみ）
- 例: `modern_study_bright`, `library_interior_quiet`, `home_livingroom_morning`, `cafe_counter_daytime`

## 参考情報の活用

以下の検索結果を必ず活用してください：

{reference_information}

### 活用方法

1. **食べ過ぎ情報**: 症状の進行、悪化セクションに反映
2. **メリット情報**: 背景情報、適切な摂取方法に反映
3. **デメリット情報**: 医学的メカニズム、真相解明に反映

以下の JSON 形式で出力してください：

```json
{
  "title": "{タイトル}",

  "story_summary": "ストーリー全体の要約（2-3文）",
  "sections": [
    {
      "section_key": "hook",
      "section_name": "冒頭フック・危機の予告",
      "purpose": "視聴者を引き込むため、最も衝撃的なシーンを見せる",
      "content_summary": "ずんだもんが体調不良で倒れるシーン。めたんとつむぎが心配する様子。どうしてこうなったのかという疑問を提示。",

      "max_lines": 10,
      "background": "home_livingroom_morning"
    },
    {
      "section_key": "background",

      "content_summary": "チョコレートの成分（カカオ、糖分、脂質）、一般的な効果（リラックス効果、抗酸化作用）、適切な摂取量について解説。",
      "min_lines": 10,
      "max_lines": 15,
      "background": "modern_study_room"
    }
    // ... 他のセクション
  ]

```

## アウトライン設計例

### 良い例（柔軟な構成）

```
1. 冒頭フック (6-10セリフ)
2. 背景情報 (10-15セリフ)
3. 摂取開始の理由 (12-18セリフ)
4. 初期の楽観期 (15-25セリフ)
5. 異変の始まり (20-30セリフ)
6. 症状の悪化 (25-35セリフ)
7. 決定的イベント (20-30セリフ)
8. 医学的解説 (15-25セリフ)
9. 回復と解決策 (10-20セリフ)

合計: 約133-208セリフ
```

### 悪い例（避けるべき）

```
1. 冒頭 (5-10セリフ) ← 内容が不明確
2. 中盤 (10-20セリフ) ← 目的が不明確
3. 終盤 (5-10セリフ) ← 構成が雑

合計: 20-40セリフ ← 少なすぎる
```

## 注意事項

1. **医学的正確性を最優先**

   - 参考情報に基づいた内容
   - 誇張しすぎない

2. **ストーリーの一貫性**

   - 各セクションが論理的につながる
   - 時系列が明確

3. **適切なボリューム**

   - 合計 130-160 セリフ程度を目安
   - 重要なセクションに重点を置く

4. **視聴者への配慮**

   - 過度な不安を煽らない
   - 建設的な解決策を提示

5. **エンターテイメント性**
   - キャラクターの個性を活かせる展開
   - 視聴者が飽きない構成
