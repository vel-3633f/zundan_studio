# 食べ物摂取過多タイトル生成プロンプト

{food_name}を毎日食べ続けた場合の動画タイトルを生成してください。

## タイトル生成の目的

視聴者の興味を引き、クリックしたくなるような煽り気味のタイトルを作成します。
医学的リスクを示唆しつつ、エンターテイメント性を重視してください。

## タイトル要件

### 必須要素

- YouTube でクリックされやすい煽り要素
- 【検証】【衝撃】【毎日食べ続けた】【危険】などのキーワードを活用
- 医学的リスクや健康への影響を示唆
- 30-50 文字程度

### 推奨される表現パターン

1. **検証系**: 【検証】〜を毎日食べ続けたら〜
2. **衝撃系**: 【衝撃】〜の恐ろしい真実
3. **警告系**: 【危険】〜を食べ過ぎると〜
4. **結果系**: 〜を 1 ヶ月食べ続けた結果...
5. **疑問系**: 〜を毎日食べるとどうなる？

### 避けるべき表現

- 過度に不安を煽る表現（「死ぬ」「命に関わる」など）
- 医学的に不正確な断定表現
- 食べ物自体を否定する表現

## 出力形式

以下の JSON 形式で出力してください：

```json
{
  "title": "YouTubeタイトル（30-50文字）",
  "food_name": "{food_name}",
  "hook_phrase": "タイトルの最も印象的な部分（10-15文字程度）"
}
```

### hook_phrase について

タイトルの中で最も視聴者の興味を引く部分を抽出してください。
例：

- タイトル: 【検証】チョコレートを毎日食べ続けたら肌が荒れまくった件
- hook_phrase: 肌が荒れまくった

## タイトル生成例

### 良い例

- 【検証】ラーメンを毎日食べ続けたら体に異変が...【衝撃の結果】
- 【危険】チョコレートの食べ過ぎで起きた恐ろしい症状とは
- 毎日カレーを食べ続けた結果、医者に警告された話

### 悪い例（避けるべき）

- ラーメンは体に悪い（煽り要素なし、断定的）
- チョコレートを食べると死にます（過度な不安煽り）
- カレー好き必見！（内容が不明確）

## 注意事項

1. 参考情報に基づいた医学的に正確な内容にすること
2. 視聴者の興味を引きつつ、誤解を招かない表現を心がけること
3. 食べ物自体を否定するのではなく、「食べ過ぎ」の問題点を強調すること
4. エンターテイメント性と教育的価値のバランスを取ること
