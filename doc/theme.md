# YouTube企画書：思考実験バラエティ「もしも〇〇だったら？」

## 1. チャンネルのコンセプト

- **ジャンル**: 思考実験・空想科学・社会シミュレーション
- **ターゲット**: 佐藤 義久（30歳・独身男性・オタク・事務職・現状に不満あり）
- **コアバリュー**: 退屈で閉塞感のある現実を、**「圧倒的な妄想」と「知的なシミュレーション」**で破壊する。「世界がこうなったら面白いのに（あるいは怖い）」というオタク特有の好奇心を満たす。

## 2. なぜ「もしも系」なのか？（採用理由）

### AIリスクの回避（ハルシネーション対策）

- 未来予測や架空のシナリオであるため、「正解」が存在しない
- AIが多少飛躍した嘘をついても、「シミュレーションの一例」としてエンタメ化できる

### ターゲットへの適合性（佐藤義久への刺さり）

- **現実逃避**: 「会社に行きたくない」「世界が終わればいい」という深層心理を刺激する
- **知識欲**: 科学的・論理的な考察（ゾンビ対策の最適解など）に対して、コメント欄で持論を語りたがるオタク気質にマッチする

## 3. 具体的なテーマ案（ネタ帳）

### カテゴリA：社会リセット・崩壊（現実逃避願望）

今の社会システムが壊れることで、カタルシスを感じるテーマ。

- もしも日本で『ゾンビパンデミック』が起きたら？（埼玉在住・30代一般男性の生存ルートを真剣に検証）
- もしも今日から『インターネット』が完全に消滅したら？（スマホ依存の現代人がどう発狂するか、インフラはどうなるか）
- もしも人間が『眠らなくていい体』になったら？（遊べる時間が増えるか、それとも24時間労働になるか）
- もしも日本から『税金』が完全になくなったら？（公共サービス停止によるカオス vs 手取り倍増の快感）

### カテゴリB：SF・オタク妄想（少年の心）

アニメやゲームの設定を現実に持ち込むテーマ。

- もしも30歳の記憶のまま『小学生』からやり直せたら？（ビットコインを買うか、同級生にマウントを取るか）
- もしも『異世界転生』が実在したら、現代知識で無双できるか？（言葉の壁、衛生問題など、リアルなハードルを突きつける）
- もしも『デスノート』が実在し、誰でも拾える状況になったら？（世界はどう混乱するか、自分ならどう使うか）

### カテゴリC：極限状態・科学（怖いもの見たさ）

極端な状況下での変化を楽しむテーマ。

- もしも地球の『酸素』が5秒間だけ消滅したら？（建物崩壊、鼓膜破裂などの物理的カオス）
- もしもこの世から『嘘』が消えたら？（建前社会の崩壊、カップルの修羅場）
- もしも『寿命』が売買できるようになったら？（いくらが相場か？ 貧困層が時間を売るディストピア）
